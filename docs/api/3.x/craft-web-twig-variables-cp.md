### `nav()`





Returns the control panel nav items.

Each control panel nav item should be defined by an array with the following keys:

- `label` â€“ The human-facing nav item label
- `url` â€“ The URL the nav item should link to
- `id` â€“ The HTML `id` attribute the nav item should have (optional)
- `icon` â€“ The path to an SVG file that should be used as the nav item icon (optional)
- `fontIcon` â€“ A character/ligature from Craftâ€™s font icon set (optional)
- `badgeCount` â€“ A number that should be displayed beside the nav item when unselected
- `subnav` â€“ A sub-array of subnav items

Subnav arrays should be associative, with identifiable keys set to sub-arrays with the following keys:

- `label` â€“ The human-facing subnav item label
- `url` â€“ The URL the subnav item should link to

For example:

```php
[
    'label' => 'Commerce',
    'url' => 'commerce',
    'subnav' => [
        'orders' => ['label' => 'Orders', 'url' => 'commerce/orders',
        'discounts' => ['label' => 'Discounts', 'url' => 'commerce/discounts',
    ],
]
```

Control panel templates can specify which subnav item is selected by defining a `selectedSubnavItem` variable.

```twig
{% set selectedSubnavItem = 'orders' %}
```
