### `getEntryById()`





Returns an entry by its ID.

```php
$entry = Craft::$app->entries->getEntryById($entryId);
```


[View source](https://github.com/craftcms/cms/blob/master/src/services/Entries.php#L35-L53)


#### Arguments

- `$entryId` ([integer](http://php.net/language.types.integer)) – The entry’s ID.
- `$siteId` ([integer](http://php.net/language.types.integer), [null](http://php.net/language.types.null)) – The site to fetch the entry in. Defaults to the current site.

#### Returns

[craft\elements\Entry](craft-elements-entry.md), [null](http://php.net/language.types.null) – The entry with the given ID, or `null` if an entry could not be found.
